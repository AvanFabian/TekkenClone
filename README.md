## <a name="quick-start">🚀 Quick Start</a>

1. Clone repository:

    ```bash
    git clone https://github.com/AvanFabian/TekkenClone.git
    ```

2. Navigate to the project directory:

    ```bash
    cd TekkenClone
    ```

3. Add Project to Unity Hub
4. Open the Unity Editor
